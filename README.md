# 抖音视频/图片批量下载工具

一款简洁高效的抖音内容批量下载工具，支持下载指定博主的视频和图片，并自动整理归类。

## 功能特性

- **批量下载**：支持同时下载多位抖音博主的内容
- **自动分类**：视频和图片自动分类存储
- **断点续传**：智能识别已下载内容，避免重复下载
- **格式转换**：图片自动从 WebP 格式转换为 JPG
- **时间命名**：使用发布时间作为文件名，方便管理
- **去重机制**：通过 URL 特征码识别重复内容

## 目录结构

```
douyin/
├── douyin.py              # 主程序
├── aweme_list.json        # 博主列表（自动生成）
├── json.log               # API 响应日志（手动导入）
├── cfg.ini                # 配置文件（如有）
└── download/              # 下载目录（如有）
    ├── {博主名称}/
    │   ├── url.txt        # 已下载 URL 记录
    │   ├── *.mp4          # 视频文件
    │   └── picture/       # 图片目录
    │       └── *.jpg      # 图片文件
    └── ...
```

## 使用方法

### 第一步：获取抖音数据

1. 打开抖音网页版 (https://www.douyin.com)
2. 登录后进入目标博主的主页
3. 按 F12 打开开发者工具，切换到 **Network** 标签
4. 向下滚动加载博主所有视频内容
5. 找到类型为 `fetch/xhr` 且包含 `aweme_list` 的请求
6. 复制 Response 内容，追加到 `json.log` 文件中（每行一条 JSON）

### 第二步：运行程序

```bash
python douyin.py
```

### 第三步：功能选择

程序启动后显示主菜单：

| 指令 | 功能 |
|------|------|
| `a` | 开始下载所有博主的内容 |
| `b` | 删除指定博主（清除已下载内容） |
| `c` | 查看博主列表 |
| `x` | 退出程序 |

## 配置说明

### 修改下载路径

在 `douyin.py` 第 128 行修改默认下载路径：

```python
download_dir = 'F:/图片/douyin/'  # 修改为你的目标路径
```

### User-Agent 配置

如遇下载失败，可在第 70 行更新浏览器标识：

```python
'User-Agent': '你的浏览器User-Agent'
```

## 环境依赖

- Python 3.x
- PIL (Pillow)

安装依赖：

```bash
pip install Pillow
```

## 注意事项

1. **数据时效性**：抖音 API 数据具有时效性，建议获取后尽快下载
2. **登录状态**：确保抖音账号已登录，否则可能无法获取完整数据
3. **下载频率**：大量下载时建议适当增加延时，避免触发风控
4. **存储空间**：视频文件较大，请确保目标磁盘空间充足
